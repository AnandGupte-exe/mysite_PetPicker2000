The PetPicker2000
How it Works

The first step is to ask the user for data and I did that by making sliders which people

can move to select there number for each question. We also need data for each animal. After we

have the user data and the animal data we need to scale the data. We do this by finding the

maximum value for each question. Then we divide each number in that question by the max

number. Once everything is scaled we plot each animal point using each question value as a

coordinate. So there are 1 dimension for each question. After that weplot the users data.

Then we find the euclidean distance () to find the distance between each point and the users

point. Then we look for the smallest distance and that is the best fitting animal.
